# Learning Programming — A Beginner's Guide

Welcome! This is a self-paced programming course built for beginners. Originally created for a family member, it walks through computer science fundamentals before diving into practical web development with JavaScript, TypeScript, and React.

No prior experience needed. Each section builds on the previous one, and every section ends with **Quiz & Checkpoints** to reinforce what you've learned. Work through them in order — the concepts stack.

---

## How to Use This Course

1. Start at Section 01 and work your way forward.
2. Read each page, then try the code examples yourself.
3. At the end of each section, complete the **Quiz & Checkpoints** before moving on.
4. Check the **Sources** page at the end of each section for further reading.
